# JS- paint

바닐라 js로 그림판 만들기

### 기간
2022.02.15~2022.02.17

### ⚙ 기능

- 캔버스에 여러 가지 색상을 이용하여 그림을 그릴 수 있다.
- 펜의 굵기를 조절할 수 있다.
- 전체 색칠/삭제를 할 수 있다.
- 그림을 png 파일로 저장할 수 있다.
- 투두를 작성하고 삭제할 수 있다.
